## Run Locally

**Prerequisites:**  Node.js

**important**
unzip the habitoff.zip file it has all necesarry folders

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
